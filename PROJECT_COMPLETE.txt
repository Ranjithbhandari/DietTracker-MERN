================================================================================
                        TROUBLESHOOTING
================================================================================

ISSUE: Backend won't start
SOLUTION: Check if port 5000 is in use
  lsof -i :5000  (macOS/Linux)
  netstat -ano | findstr :5000  (Windows)

ISSUE: Frontend won't load
SOLUTION: Clear cache and hard refresh
  Ctrl+Shift+Delete (cache)
  Ctrl+Shift+R (hard refresh)

ISSUE: MongoDB connection error
SOLUTION: Verify connection string and IP whitelist
  Check .env file
  Check MongoDB Atlas settings

ISSUE: Login not working
SOLUTION: Check backend is running and .env is correct
  curl http://localhost:5000/api/health
  Verify VITE_API_URL in frontend .env

For more help, see LOGIN_TROUBLESHOOTING.md
