================================================================================
                        QUICK START
================================================================================

BACKEND SETUP:
  1. cd backend
  2. npm install
  3. cp .env.example .env
  4. Edit .env with MongoDB URI and JWT secret
  5. npm start

FRONTEND SETUP (New Terminal):
  1. cd frontend
  2. npm install
  3. cp .env.example .env
  4. npm run dev

VISIT:
  Frontend: http://localhost:5173
  Backend: http://localhost:5000

RUN TESTS:
  cd backend
  npm test

================================================================================
                        DEPLOYMENT
================================================================================

BACKEND (Render):
  1. Push to GitHub
  2. Create Web Service on Render
  3. Set environment variables
  4. Deploy

FRONTEND (Vercel):
  1. Push to GitHub
  2. Import project on Vercel
  3. Set VITE_API_URL environment variable
  4. Deploy

DATABASE (MongoDB Atlas):
  1. Create free cluster
  2. Create database user
  3. Whitelist IP (0.0.0.0/0 for development)
  4. Get connection string

See DEPLOYMENT_GUIDE.md for detailed instructions.

================================================================================
                        API ENDPOINTS
================================================================================

AUTHENTICATION:
  POST /api/auth/register          Register new user
  POST /api/auth/login             Login user

USER PROFILE:
  GET /api/user/profile            Get user profile (protected)
  PUT /api/user/profile            Update profile (protected)

MEALS:
  POST /api/meals                  Create meal (protected)
  GET /api/meals/today             Today's meals (protected)
  GET /api/meals/history           Last 7-10 days (protected)
  DELETE /api/meals/:id            Delete meal (protected)
